## そう
Positive:
- Verb［ます］+ そう + だ
- ［い］Adjective［い］+ そう + だ
- ［な］Adjective + そう + だ

Negative:
- Verb［ない］+ な + そう + だ
- ［い］Adjective ［ない］ + なさ + そう + だ
- ［な］Adjective + では(1)ない + なさ + そう + だ

- Exceptions : いい ￫ よい + さそう + だ ￫ よさそうだ
